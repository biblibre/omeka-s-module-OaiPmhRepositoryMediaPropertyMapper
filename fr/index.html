Introduction

OaiPmhRepository Media Property Mapper est un module compagnon pour OaiPmhRepository. Il permet d’exposer des propriétés des médias comme si elles faisaient partie du contenu.

Comment ça marche ?

Ça utilise l’évènement oaipmhrepository.values.pre pour ajouter des valeurs à l’ensemble des valeurs qui seront ensuite traitées par le module OaiPmhRepository. Cela signifie que:

  • Les valeurs peuvent seulement être ajoutées. Elles ne remplacent pas de valeurs existantes

  • Selon le format de métadonnées utilisé (oai_dc, mets, …), seul un sous-ensemble de ces valeurs sera réellement exposé. Par exemple, oai_dc expose uniquement les propriétés dcterms.
